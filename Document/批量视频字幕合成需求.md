## 背景
+ 现在有一批没有字幕的原始视频和多语言的外挂字幕，需要进行批量合成
+ 原始视频放在一个文件夹，多语言字幕放在另一个文件夹
+ 多语言字幕文件夹中，又分为很多个国家的语种文件夹，每个文件夹下存放了对应语种的多语言
+ 现在需要将这一批原始视频，分别合成为带不同国家字幕的视频文件
+ 资源的目录结构为：
  + 原始视频
    + 001.mp4
    + 002.mp4
    + ....
  + 字幕文件
    + AR
      + 001_AR.str
      + 002_AR.str
      + ....
+ 期望最终输出合成后的视频文件结构为：
  + 输出视频文件
    + AR
      + 001_AR.mp4
      + ....
    + CN
      + 001_CN.mp4
    + ....

## 需求
+ 需要一个前端的交互界面
  + 需要用户自己选择原视频的文件路径
  + 选择str的文件路径
    + 自动分析str文件夹中，有哪些语种的文件
  + 点击批量合成后，原视频分别和不同语种的字幕文件合并，最终输出带字幕的视频
  + 最终输出文件严格按照上方给出的命名保存在文件夹中
