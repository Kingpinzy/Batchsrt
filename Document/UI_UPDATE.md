# 界面布局更新说明

## 更新内容

### 布局变更 ✅

从**上下布局**改为**左右分栏布局**，提升操作效率和视觉体验。

## 新布局设计

### 整体结构

```
┌────────────────────────────────────────────────────┐
│                     页面标题                         │
├─────────────────────┬──────────────────────────────┤
│                     │                              │
│    左侧配置面板      │       右侧进度监控            │
│   (480px宽度)       │      (自适应宽度)             │
│                     │                              │
│  📁 配置参数         │    📊 实时进度                │
│  - 原视频文件夹      │    - 状态徽章                 │
│  - 字幕文件夹        │    - 当前任务                 │
│  - 输出文件夹        │    - 进度条                   │
│  - 语种标签          │    - 详细日志                 │
│                     │                              │
│  ⚡ GPU加速选项      │                              │
│  💡 使用指南         │                              │
│  🚀 开始/终止按钮    │                              │
│                     │                              │
└─────────────────────┴──────────────────────────────┘
```

### 左侧面板（配置区）

**宽度**: 480px 固定宽度
**背景**: 浅灰色 (#fafbfc)
**内容**:
- 📁 配置参数标题
- 原视频文件夹输入 + 验证按钮
- 字幕文件夹输入 + 验证按钮
- 输出文件夹输入 + 验证按钮
- 检测到的语种标签
- GPU硬件加速选项
- 使用指南提示
- 开始/终止操作按钮

**特点**:
- 独立滚动区域
- 集中的配置管理
- 清晰的视觉分组

### 右侧面板（监控区）

**宽度**: 自适应剩余空间
**背景**: 白色
**内容**:
- 📊 实时进度标题
- 状态徽章（待机/处理中/完成/错误）
- 当前处理任务显示
- 进度条（百分比显示）
- 任务统计（已完成/总数）
- 实时日志控制台

**特点**:
- 全高度日志区域
- 实时更新显示
- 专注的进度监控

## 视觉优化

### 1. 颜色和间距
- **左侧面板**: 浅灰背景区分功能区域
- **右侧面板**: 白色背景突出日志内容
- **边框**: 细线分隔（1px #e2e8f0）
- **圆角**: 统一使用12-16px圆角
- **间距**: 模块间距20-24px

### 2. 排版优化
- **标题层级**: 使用emoji + 粗体标题
- **字体大小**:
  - 区域标题: 16px
  - 表单标签: 14px
  - 日志文本: 13px
- **行高**: 1.6-1.8倍提升可读性

### 3. 交互反馈
- **输入框**: 聚焦时显示蓝色边框和阴影
- **按钮**: hover时上浮2px + 阴影增强
- **验证状态**: 绿色/红色边框 + 背景色
- **滚动条**: 自定义样式，圆角半透明

## 响应式设计

### 大屏幕 (>1200px)
- 左右分栏布局
- 左侧480px，右侧自适应
- 最佳体验

### 中等屏幕 (768px - 1200px)
- 自动切换为上下布局
- 左侧面板在上，宽度100%
- 右侧面板在下，宽度100%
- 保持功能完整

### 小屏幕 (<768px)
- 单列布局
- 输入框和按钮垂直排列
- 减小内边距优化空间
- 完全响应式

## 技术实现

### CSS Flexbox布局
```css
.content {
    display: flex;
    flex: 1;
    overflow: hidden;
}

.left-panel {
    width: 480px;
    flex-shrink: 0;
    overflow-y: auto;
}

.right-panel {
    flex: 1;
    display: flex;
    flex-direction: column;
}
```

### 独立滚动区域
- 左侧面板：独立垂直滚动
- 右侧日志：独立垂直滚动
- 顶部标题：固定不滚动

### 全高度适配
```css
.container {
    height: calc(100vh - 80px);
    display: flex;
    flex-direction: column;
}
```
